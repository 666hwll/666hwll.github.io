# Shortcuts KiCad (D. Zmugg):

1. A: Komponenten hinzufügen
2. M: Bauteil verschieben
3. R: Bauteil rotieren
4. G: Bauteil schieben (zieht Bauteil mit Anschlussleitungen)
5. N: Raster ändern (Achtung kann Probleme erzeugen!)
6. T: Text hinzufügen
7. L: Label
8. E: Eigenschaften Bauteil (bei Footprints Wildcards möglich wie R?)



##### Selektieren mit Rahmen:

--> Von links nach rechts - alles was ganz drinnen ist

--> von rechts nach links - alles was berührt wird



##### Mehrere Komponenten in einem Package - selektieren - rechte Maustaste



##### Label(/Netzbezeichnung) auf Netz platzieren:

Empfehlung: ans Ende der Leitung setzen und dann mir 'R' rotieren

Bezeichner --> ~{RESET}

--> RESET invertiert!!!



##### PCB-LAYOUT:

1. P: Pack and Move
2. G: schieben
3. M: move
4. V: Via setzen
5. X: routen
6. +-: Layer wechseln



##### Leiterbahnklassen (für unterschiedliche Leiterbahnbreiten)

--> während dem Routen



##### Eine ganz geroutete Leitung löschen:

--> ein Segment markieren

--> Mit 'U' weiterselektieren bis alles markiert, dann löschen



##### Nachträgliche Korrekturen im Schaltplan - Bauteil enfernt -->

--> Platine aus Schaltplan aktualisieren -> Footprints aktualisieren -> anklicken (Footprints entfern, wenn ihnen kein ...)



##### Bestellvorgang:

--> Kontrollieren, ob PIN1 auch PIN1 ist!!!

--> PlugIns (zb.: Fabrication Toolkit) installieren

--> 'Basic' ist günstig und gut erhältlich; Promotional Extended für kurze Zeit; Extended ist teurer
